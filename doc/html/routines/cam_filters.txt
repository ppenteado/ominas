CAM_FILTERS


 PURPOSE:
	Returns the filter name(s) for a given camera descriptor.


 CATEGORY:
	NV/LIB/CAM


 CALLING SEQUENCE:
	filters = cam_filers(cd, index)


 ARGUMENTS:
  INPUT:
	cd:	 Camera descriptor.

	index:	 Index of filter to return.  If not given, all are returned.

  OUTPUT: NONE


 KEYWORDS:
  INPUT:  NONE

  OUTPUT: NONE


 RETURN:
	Names of requested filters.


 STATUS:
	Complete

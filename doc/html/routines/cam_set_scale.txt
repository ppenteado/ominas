CAM_SET_SCALE


 PURPOSE:
       Replaces the scale array for each given camera descriptor.


 CATEGORY:
	NV/LIB/CAM


 CALLING SEQUENCE:
	cam_set_scale, cd, scale


 ARGUMENTS:
  INPUT:
	cd:	Array (nt) of CAMERA descriptors.

	scale:	Array (2,nt) of new scale values.

  OUTPUT: NONE


 KEYWORDS:
  INPUT:  NONE

  OUTPUT: NONE


 RETURN: NONE


 STATUS:
	Complete

 	Written by:	Spitale, 1/1998

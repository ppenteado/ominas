RNG_DESC


 PURPOSE:
	Returns the description string for each given ring descriptor.


 CATEGORY:
	NV/LIB/RNG


 CALLING SEQUENCE:
	desc = rng_desc(rd)


 ARGUMENTS:
  INPUT: 
	rd:	 Array (nt) of RING descriptors.

  OUTPUT: NONE


 KEYWORDS:
  INPUT: NONE

  OUTPUT: NONE


 RETURN:
	Description string associated with each given ring descriptor.


 STATUS:
	Complete

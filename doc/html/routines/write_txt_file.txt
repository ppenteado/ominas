WRITE_TXT_FILE


 PURPOSE:
	Writes text to a file.


 CATEGORY:
	UTIL


 CALLING SEQUENCE:
	write_txt_file, fname, text


 ARGUMENTS:
  INPUT:
	fname:	Name of file to read.

	text:	Text to write to the file.

  OUTPUT: NONE


 KEYWORDS:
  INPUT:
	append:	If set, the text will be appended if the file already exists.

  OUTPUT: NONE


 RETURN: NONE


 STATUS:
	Complete


 SEE ALSO:
	read_txt_file
